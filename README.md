# Manim Auto-Renderer
Auto-rendering Manim animations via GitHub Actions.

## Available Scenes
- **pythagorean** — Pythagorean Theorem visual proof with animated right triangle
- **llm** — How Large Language Models work — token flow visualization
- **sorting** — Bubble Sort algorithm visualization with animated bars
- **fourier** — Fourier Series — sine waves building a square wave
- **gravity** — Gravity and projectile motion simulation

## Usage
Run locally: `python github_render.py "pythagorean"`
